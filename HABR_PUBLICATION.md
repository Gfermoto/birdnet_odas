## Что делать перед публикацией

### 1. Загрузить изображения на Хабр

При создании новой публикации на Хабре:
1. Открыть редактор статьи
2. Загрузить все изображения через кнопку "Добавить изображение"
3. Заменить пути в статье на URL от Хабра

**Изображения для загрузки:**

- `images/s2.jpg` - Рис. 8: Спектрограмма после обработки
- `images/photo_7.jpg` - Рис. 9: Dashboard BirdNET-Go

### 2. Заменить пути к изображениям

После загрузки изображений на Хабр, заменить в статье:
```markdown
![Описание](images/photo_2.jpg)
```
на:
```markdown
![Описание](https://habrastorage.org/webt/xx/xx/xx/xxxxxxxxxx.jpg)
```

### 4. Проверить форматирование

Хабр поддерживает Markdown, но проверьте:
- ✅ Код в тройных бэктиках работает корректно
- ✅ Математические формулы отображаются (возможно потребуется заменить на LaTeX)
- ✅ Все ссылки активны
- ✅ Изображения с подписями отображаются правильно

### 5. Тег `<cut />`

Тег `<cut />` уже добавлен после введения. Он сворачивает длинную статью на главной странице Хабра.

## Особенности Хабра

1. **Математические формулы** - если они не отображаются, оберните в `$...$` для inline или `$$...$$` для блоков
2. **Код** - используйте тройные бэктики с указанием языка: ` ```python `
3. **Подзаголовки** - ## для разделов, ### для подразделов
4. **Ссылки** - обычный Markdown формат `[текст](url)`
